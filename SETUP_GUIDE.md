# Human Typer — First-Time Setup

Welcome, and thanks for your purchase! This guide gets you up and running in about
**2 minutes**. You only do this setup **once**.

**You received:**
- The **Human Typer** app

Jump to your system: **[macOS](#macos)** · **[Windows](#windows)**

---

## macOS

### 1. Move the app (optional)
Drag **Human Typer.app** into your **Applications** folder.

### 2. Open it the first time
Because Human Typer is an independent (non-App-Store) app, macOS asks you to
confirm the first launch. This is normal and only happens once.

- **Right-click** (or Control-click) **Human Typer** → choose **Open**.
- A dialog says *"macOS cannot verify the developer…"* → click **Open**.

> **On macOS Sequoia (15) or newer:** double-click the app first (it'll be
> blocked), then go to  **Apple menu → System Settings → Privacy & Security**,
> scroll down to the message *"Human Typer was blocked"*, and click
> **Open Anyway**. Enter your Mac password if asked.

### 3. Allow it to type (Accessibility)
Typing into other apps needs one permission.

1. **Apple menu → System Settings → Privacy & Security → Accessibility.**
2. Find **Human Typer** in the list and turn the switch **ON**.
   - If it isn't listed, click the **+** button, open your **Applications**
     folder, select **Human Typer**, and add it — then turn it ON.

### 4. Allow the Stop key (Input Monitoring)
This powers the **Esc** emergency-stop.

1. In the same **Privacy & Security** window, open **Input Monitoring**.
2. Turn **Human Typer** **ON** (add it with **+** if needed).

### 5. Restart the app
**Quit** Human Typer (⌘Q) and **open it again** so the permissions take effect.

✅ **You're done.** See **[How to use it](#how-to-use-it)** below.

## Windows

### 1. Unzip
Right-click the downloaded `.zip` → **Extract All**. Open the extracted
**HumanTyper** folder.

### 2. Run it the first time
Double-click **HumanTyper.exe**. Windows may show
*"Windows protected your PC"* (because the app is independent). This is normal:

- Click **More info** → **Run anyway**.

> If nothing opens, install Microsoft's free **Edge WebView2 Runtime**
> (most PCs already have it): search "WebView2 Runtime download", install, retry.

✅ **You're done.**

## How to use it

1. **Paste your text** into the big box.
2. **Set the speed** — lower = faster. The default (~100 ms) types like a person;
   slide left for quick, right for relaxed.
3. *(Optional)* Toggle **Humanize** for natural rhythm, or off for steady speed.
4. Click **Start Typing**, then **click into the app/field** you want it typed
   into (a browser box, a document, anywhere).
5. After a short **countdown**, it types for you.
6. **Press `Esc` anytime to stop instantly** — even if another app is in front.

## Troubleshooting

**"It shows it's typing, but nothing appears."**
The Accessibility permission isn't active yet. Re-check **Step 3**, make sure
**Human Typer** is **ON** in Accessibility, then **quit and reopen** the app.

**The text goes into the wrong place.**
After clicking **Start**, click into the exact field you want *before* the
countdown ends. Whatever is focused when typing begins is where it goes.

**"The app is damaged / can't be opened."**
Re-download the file (it can corrupt during transfer), then follow **Step 2**.
